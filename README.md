Nama Aplikasi

MotoMaintenance

Deskripsi Aplikasi

MotorMaintenance adalah aplikasi manajemen perawatan sepeda motor berbasis desktop yang dirancang khusus untuk membantu pemilik kendaraan memantau kondisi teknis dan pengeluaran operasional secara real-time.
Aplikasi ini mengintegrasikan pencatatan aktivitas harian (seperti jarak tempuh dan konsumsi BBM) dengan sistem pemantauan kesehatan komponen (seperti oli mesin, oli gardan, ban, dan kampas rem) melalui indikator visual yang intuitif. Dengan menggunakan pendekatan berbasis data, MotorMaintenance mampu memberikan peringatan dini sebelum komponen mencapai batas usia pakainya, sehingga mencegah kerusakan fatal dan menjaga performa kendaraan tetap optimal. 

Cara Menjalankan

Clone Repo: git clone https://github.com/inyomanswardijayaputra/pv26-miniproject-motomaintenance-f1d02310057.git
Install Library : pip install PySide6
Running Aplikasi/Python : python main.py

Teknologi yang Digunakan

Bahasa Pemrograman: Python
Framework GUI: PySide6 / PyQt6 (Qt for Python)
Database: SQLite
Styling: QSS (Qt Style Sheets)
